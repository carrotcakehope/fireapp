기능 소개

🧯
소방시설탐색기
건물 용도·면적·층수 등을 입력하면 현행법 기준으로 설치 의무가 있는 소방시설을 자동으로 도출합니다. 다중이용업소 안전시설도 함께 확인할 수 있습니다.
📅
날짜 계산기
자체점검 제출기한, 안전관리자 선임 기한, 부적합 조치기한을 자동으로 계산합니다. 결과를 메인화면 알림판에 추가해두면 기한을 놓치지 않을 수 있습니다.
🏢
작동·종합 대상 판독기
스프링클러 설치 여부, 연면적, 층수 등의 조건에 따라 작동기능점검 또는 종합정밀점검 대상인지를 판정합니다.
👥
다중이용업소 판독기
업종을 선택하고 질문에 답하면 다중이용업소 해당 여부를 판정합니다. 해당 시 설치해야 하는 안전시설도 안내됩니다.
🧮
수용인원 계산기
건물 용도와 면적을 입력하면 소방관계법령에 따른 법정 수용인원을 자동 산정합니다.

앱처럼 설치하기

📲 바탕화면·홈 화면에 바로가기 추가
메인화면 하단의 바로가기 추가 버튼을 누르면 기기에 맞는 방법을 안내해 드립니다.

PC — .url 바로가기 파일을 다운로드해서 바탕화면에 놓으면 더블클릭으로 바로 열 수 있습니다.
안드로이드 — Chrome에서 "홈 화면에 추가"하면 앱 아이콘이 생성됩니다.
아이폰 — 사파리 하단 공유버튼 → "홈 화면에 추가"를 탭하세요.
✅ 앱으로 설치하면 좋은 점
주소창 없이 전체화면으로 실행되고, 한 번 로딩된 후에는 인터넷이 없어도 기본 기능을 사용할 수 있습니다.

화면 설정

🌙 다크 / ☀️ 라이트 모드
화면 우측 상단의 아이콘 버튼으로 다크모드와 라이트모드를 전환할 수 있습니다. 선택한 테마는 기기에 저장됩니다.

메인화면 알림판

📌 날짜 계산 결과 저장
날짜 계산기에서 계산 후 "메인화면에 표시" 버튼을 누르면 메인화면 오른쪽 알림판에 기한이 표시됩니다. D-day가 7일 이내로 다가오면 빨간색으로 강조됩니다.
